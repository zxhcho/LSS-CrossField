# Learning Sparse Singularities for Cross Field Design

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Paper](https://img.shields.io/badge/Paper-TOG-red)](https://dl.acm.org/doi/10.1145/3787520)

Official implementation of the paper **"Learning Sparse Singularities for Cross Field Design"**.

This repository contains the neural-analytical hybrid pipeline for generating problem-aware high-quality cross fields by predicting sparse singularities on 2D surface meshes.
